Primeiro passo:
Verificar o chrome driver do seu Chrome
https://googlechromelabs.github.io/chrome-for-testing/
- Baixar Chrome driver
- Trocar caminho do chrome driver
- Depois troque os caminhos do seu computador

Trocar os locais:
